Project Details

Developed a Road accident sensor kit (aka RAKSHA) for cars that sends passenger information and location details to the nearest police stations and hospitals when an accident occurs for faster assistance.

Skills Developed: - Safety-Critical Embedded System and Algorithm Design
Key Achievement:- This project is selected and sponsored by NAIN, Govt. of Karnataka.
Used: - Arduino Programming, Eagle CAD, RFID (RC552), and GPS (NEO-6M).
